# Retail Sales Data Analysis (SQL & Python)

## Overview
This project demonstrates practical data analysis skills using both SQL and Python.  
It focuses on extracting business insights from retail sales data through relational database queries and exploratory data analysis.

---

## Tools & Technologies
- SQL (SQLite)
- Python (Pandas, Matplotlib)

---

## SQL Analysis

### Database Design
Relational database with the following tables:
- customers
- products
- sales

### Key Analysis
- Monthly revenue trends
- Top products by revenue
- Revenue by city
- Average order value (AOV)
- Top customers by lifetime spend
- Customer segmentation using CASE statements

### How to Run
1. Execute the SQL schema and data script
2. Run the analysis queries to generate insights
