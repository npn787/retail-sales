# Retail Sales SQL Analysis

## Overview
This project analyzes retail sales data using a relational database to extract business insights such as revenue trends, top products, and customer segmentation.

## Tools
- SQL (SQLite)

## Database Design
Tables:
- customers
- products
- sales

## Key Analysis
- Monthly revenue trends
- Top products by revenue
- Revenue by city
- Average order value (AOV)
- Top customers by lifetime spend
- Customer segmentation using CASE statements

## How to Run
1. Execute the schema and data script
2. Run analysis queries from the queries file
